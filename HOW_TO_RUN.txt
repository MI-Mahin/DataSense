-----------------------------

1. Open your web browser (Chrome/Edge/Firefox)

2. Go to: http://localhost:3000

   Example questions:
   - Show all distributors
   - List all products
   - Calculate total revenue
   - Show recent orders


TROUBLESHOOTING
===============

Problem: "Port already in use"
Solution: Kill the process using that port or change the port number

Problem: "Cannot connect to database"
Solution: Check .env file in nl2sql folder has correct credentials

Problem: "Frontend can't connect to backend"
Solution: Make sure backend is running at http://localhost:5001

Problem: "Module not found"
Solution: Run "pip install -r requirements.txt" in backend
         Or "npm install" in frontend


NOTES
=====

- Backend runs on port 5001
- Frontend runs on port 3000
- Both must be running simultaneously
- Do NOT close the terminal windows while using the app
- To stop: Press Ctrl+C in each terminal window
