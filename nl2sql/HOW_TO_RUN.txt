====================================================================
  DATASENSE - NL to SQL with Query Execution
====================================================================

DESCRIPTION:
-----------
This script converts Natural Language queries to SQL and optionally 
executes them on the datasense MySQL database.

Features:
- Natural Language to SQL conversion using LLM
- Query validation for safety (read-only)
- Query execution on MySQL database
- Formatted table results


PREREQUISITES:
-------------
1. Python 3.7 or higher installed on your system
2. MySQL database accessible at 10.101.13.28:6507
3. Ollama server running on http://192.168.11.10:11434
4. Database credentials configured in .env file


HOW TO SET UP:
-------------

Step 1: Open PowerShell or Command Prompt

Step 2: Navigate to the project directory
   cd c:\Users\Legion\Desktop\Internship\data-sense\nl2sql

Step 3: Create a virtual environment
   python -m venv venv

Step 4: Activate the virtual environment
   .\venv\Scripts\activate

Step 5: Install the required packages
   pip install -r requirements.txt


HOW TO RUN:
-----------

Step 1: Activate virtual environment (if not already activated)
   .\venv\Scripts\activate

Step 2: Run the script
   python nl_to_sql.py

Step 3: Enter your natural language query when prompted

Step 4: Review the generated SQL query

Step 5: Choose whether to execute it (yes/no)

Step 6: View results in formatted table

Step 7: Type "exit" to quit


EXAMPLE USAGE:
-------------

Enter your question: Show me all active distributors
Generated SQL:
SELECT * FROM distributors WHERE is_active = TRUE;

Execute this query? (yes/no): yes

Results displayed in table format with row count.


SAFETY FEATURES:
---------------
- Only SELECT queries are allowed (read-only mode)
- Query validation before execution
- Error handling for invalid queries
- Connection testing at startup


NOTES:
-----
- Database credentials are stored securely in .env file
- .env file is ignored by git for security
- All queries are validated before execution
- Only read operations are permitted
